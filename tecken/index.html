Teckenkodning

En crash-course av Stickheadz32.

Vad är teckenkodning?

Inom datorvärlden används teckenkodning till att representera tecken och symboler med hjälp av respektive kodssystem.

Exempel

Inom HTML:

<meta charset="ISO-8859-1" /> <meta http-equiv="Content-Type" content="text/html; charset=ISO-8859-1" />

Inom CSS:

@charset "ISO-8859-1";

Inom Javascript:

string test = "\u00E4"

För att exempelvis blinda personer ska kunna förstå webbsidans innehåll, då kan man - inom HTML - omringa textstycke med ett abbr-tagg med en title-attribut som beskriver textstycket.

<p>Beskriv en kort, <abbr title="Synonym till kortfattad; kort beskriven">koncis</abbr> och korrekt sammanfattning.</p>
