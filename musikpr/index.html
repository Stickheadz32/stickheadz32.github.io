Testing out the MediaElement.js plugin

HTML5 <audio> and <video> was supposed to be awesome, powerful and fun.

But browser vendors couldn't agree on a codec. Thus, older browsers (specifically those with no HTML5 support), don't support <audio> or <video> at all, which means that <audio src="myfile.mp3" /> doesn't work, nor does <video src="myfile.mp4 />.

This plugin, however, - MediaElement.js, made by John Dyer - is a Javascript plugin that replaces <audio> and <video> tags with a native HTML+CSS player that looks and works the same in every browser.

<audio> tag in action with the plugin. Example song: "Go Down" by Alexander Brandon, featured in Unreal Tournament (1999).

<video> tag in action. This video is a replica of the audio message user interface from the video game Deus Ex (2000).
